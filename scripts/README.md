# Scripts

Small utility scripts used for development and troubleshooting.

- `scripts/test_imports.py`: quick sanity check that core imports + tool registration work
- `scripts/test_tool.py`: tool execution smoke tests
- `scripts/transcribe.py`: ad-hoc audio transcription helper
- `scripts/fix_piper_espeak_master.sh`: installs espeak-ng master for Piper compatibility (see `docs/notes/TTS_FIX_NEEDED.md`)
